
# Taiwan Trip Timeline App V2

## Features
- Day-by-day itinerary

- Google Maps embed
- Clickable image gallery
- Weather section
- Elderly-friendly notes
- Responsive design

## Deploy GitHub Pages

1. Create repository
2. Upload files
3. Settings → Pages
4. Deploy from branch
5. Select main/root
6. Save
