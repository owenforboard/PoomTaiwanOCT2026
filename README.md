# Taiwan Trip Interactive HTML + JSON App

## Files
- index.html
- trip.json

## Run Locally
Open:
index.html

or use:
npx serve

## Deploy on GitHub Pages

1. Create GitHub repository

2. Upload:
- index.html
- trip.json

3. Go to:
Settings → Pages

4. Select:
Deploy from branch

5. Choose:
main / root

6. Save

Your site will be available at:
https://YOURNAME.github.io/REPO_NAME

## Deploy on Netlify
Drag the folder into:
https://app.netlify.com/drop
